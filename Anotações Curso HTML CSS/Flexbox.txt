
Objetivos da aula 


- Conhecer a estrutura básica

- Entender a diferença entre flex Container e Flex item 

- Conhecer inicialmente alguma propriedade


	Suporte

O FlexBox foi projetado como um modelo de layout unidemensional e como um método que pode oferecer distribuição de espaço entre itens em uma interface e recursos de alinhamento. 


	Flex Container 

É a tag que envolve os itens, será nela que iremos aplicar a propriedade "display:flex". Transforma todos os seus itens filhos em flex itens.

obs: pode ser feita em qualquer tag "div, h1,h2, span, a..".

	
Propriedades relacionadas:

- Display = Inicializador do Container

- Flex-direction = Direcionamento dos Itens

- Flex-wrap = Aplica para a quebra de linha ou não.

- Flex-flow = Abreviação para direction ou wrap.

- Justify-content = Alinha os itens do container de acordo com sua direção

- Align-items = Alinha os itens de acordo com o eixo do container

- Align-content = Alinha as linhas desse container.


	Flex Item

São os elementos filhos diretos do Flex Container. E também podem se tornar Flex Container.

Propriedades relacionadas:

- Flex-grow = Defini o fator de crescimento.

- Flex-basis = Defini o fator inicial do item antes da distribuição

- Flex-shrink = Defini a capacidade de redução

- Flex = Abreviação 

- Order = Ordem de distribuição

- Align-self = Alinhamento de um item especifico dentro do container.
