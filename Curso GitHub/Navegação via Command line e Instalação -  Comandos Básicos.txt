- Deletar pastas/arquivos


Windows commands

- cd = Possibilita que haja navegação pelas pastas, abre as pastas.

- dir = Listas, se situar dentro de um sistema operacional. 

- mkdir = Criar diretorio, criador de pasta.

- del = deleta arquivos dentro da pasta.
- rmdir = deleta toda a pasta e os arquivos dentro.

- cls = Limpar o terminal.

- Tab (atalho)  = Ele completa o nome de pastas e arquivos.
